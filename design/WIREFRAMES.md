# UniFi-Doordeck Bridge - Wireframes

## Screen Dimensions & Layout Grid

**Window Size:**
- Minimum: 1024x768px (supports older displays)
- Default: 1280x800px (comfortable for most users)
- Maximum: Responsive, scales to full screen

**Layout Grid:**
- 12-column grid system
- Gutter: 16px
- Margin: 24px (sides), 16px (top/bottom)
- Content max-width: 1400px (centered for ultra-wide displays)

## Component Specifications

### Typography Scale
- H1 (Page Titles): 32px, Semibold
- H2 (Section Headers): 24px, Semibold
- H3 (Card Titles): 20px, Medium
- Body Large: 16px, Regular
- Body: 14px, Regular
- Body Small: 12px, Regular
- Caption: 11px, Regular

### Spacing Scale
- XXS: 4px
- XS: 8px
- S: 12px
- M: 16px
- L: 24px
- XL: 32px
- XXL: 48px

- Background Secondary: #2D2D2D

### Interactive States
- Default: Base color
- Hover: 10% lighter (dark) / 10% darker (light)
- Active: 20% lighter (dark) / 20% darker (light)
- Focus: 2px outline in accent color, 2px offset
- Disabled: 50% opacity

### Animation Timing
- Fast: 100ms (button hovers, small UI changes)
- Medium: 200ms (panel transitions, tooltips)
- Slow: 300ms (page transitions, modals)
- Easing: ease-in-out (default), ease-out (entrances), ease-in (exits)

## Accessibility Specifications

### Keyboard Navigation
- Tab order follows visual hierarchy
- All interactive elements keyboard accessible
- Focus indicators always visible (WCAG 2.4.7)
- Skip links for main navigation
- Escape closes modals/dialogs
- Arrow keys navigate lists and tabs
- Space/Enter activate buttons and toggles

### Screen Reader Support
- Semantic HTML elements
- ARIA labels for all interactive elements
- ARIA live regions for status updates
- Alt text for all icons (or aria-hidden if decorative)
- Role attributes for custom components
- Announced loading states
- Clear form labels and error messages

### Color Contrast
- Text (Normal): 4.5:1 minimum (WCAG AA)
- Text (Large 18px+): 3:1 minimum (WCAG AA)
- UI Components: 3:1 minimum (WCAG AA)
- Status indicators: Don't rely on color alone (use icons + text)
- High contrast mode support

### Responsive Touch Targets
- Minimum: 44x44px (WCAG 2.5.5)
- Recommended: 48x48px
- Spacing: 8px minimum between targets

## Responsive Behavior

### Breakpoints
- Small: 1024-1280px (reduce sidebar, stack cards)
- Medium: 1280-1600px (default layout)
- Large: 1600px+ (expand content area, more columns)

### Window Resize Behavior
- Minimum window size: 1024x768px
- Sidebar collapses to icons at <1200px width
- Cards stack vertically at <1100px width
- Tables scroll horizontally at narrow widths
- Maintain aspect ratios for dashboard widgets

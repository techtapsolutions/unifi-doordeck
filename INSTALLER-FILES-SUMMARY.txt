BUILD COMMANDS:
===============

npm run build:installer              # Full build (recommended)
npm run build:installer:verbose      # With detailed output

npm run verify:install               # Post-install verification
npm run verify:install:repair        # With automatic repair

npm run upgrade:check                # Check for upgrades
npm run upgrade:simulate             # Simulate upgrade (no changes)
npm run upgrade:rollback             # Rollback to previous version
